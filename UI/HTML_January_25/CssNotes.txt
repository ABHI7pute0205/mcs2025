css Gradient property : 
    display the smooth transition between two or more specific color.

Define 3 typesof the Gradient
1. Linear Gradient : a color you want to render smooth transition.
Syntax : 
    background-image:Linear-Gradient(direction,colors[color1,2,3,4...n]); minimum 2 colors need to pass 
_____________________________________ 
2. Radical Gradient -> 
  Radial gradient is defined by its center and if we want to define radial gradient you must define at least two colors.

Syntax: 
background-image:radial-gradient(shape size at position,start color… last-color);
_____________________________________
3.conic Gradient -> 
  A conic gradient is gradient with color transitions rotated around the center points.

Syntax: background-image:conic-gradient([from angle],[at position],color [degree],color [degree]…….);
